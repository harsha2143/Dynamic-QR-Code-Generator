# Dynamic-QR-Code-Generator
- QR Code Generator with JavaScript
 
    A simple and dynamic QR Code Generator web application built using HTML, CSS, and JavaScript. This project allows users to input text or URLs and instantly generate a QR code for that input without the need for page refresh. The application provides a clean and user-friendly interface with responsive design.

- ## Features
  
    Generate QR codes dynamically based on user input (text or URL).
  
    No page refresh required for generating new QR codes.

    Error handling for empty input fields.

    Smooth animations and transitions for the QR code display.

    Simple and minimalist UI design.

- ## Technologies Used
  
    HTML5: Markup language for structuring the web page.

    CSS3: Styling for the page layout, animations, and responsive design.

    JavaScript: Handles dynamic QR code generation and error handling.

    QR Code API: Uses https://api.qrserver.com/v1/create-qr-code/ for generating QR codes.

- ## How It Works
  
    Enter a text or URL in the input field.

    Click the "Generate QR Code" button.

    The QR code corresponding to the input text/URL is displayed below the input field.

    If the input field is empty, the app will highlight the input box to prompt for user input.
